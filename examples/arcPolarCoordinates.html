ensure we always interpolate the angle over the smaller difference

 
  if (abs(theta-mousePos.y)>PI) {
    if (theta>mousePos.y) {
      theta-=TWO_PI;
    } 
    else {
      mousePos.y-=TWO_PI;
    }
  }
¶

interpolate to the new angle, adaptive speed based on current velocity
